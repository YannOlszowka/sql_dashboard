# SQL Sales Analysis & Dashboard

## Objective
Analyze retail sales data using SQL and create a dashboard to highlight key business insights.

## Dataset
Sample Superstore dataset (sales, profit, categories, regions).

## Tools
- SQLite
- SQL
- Python (Pandas)
- Power BI

## Project Structure
- `data/` → raw dataset
- `notebook/` → SQL analysis using Python
- `sql/` → SQL queries
- `dashboard/` → Power BI dashboard

## Analysis Performed
- Sales aggregation by category
- Profit analysis by region
- Profit distribution by sub-category
- Identification of low-performing segments

## Key Insights
- Technology generates the highest sales, indicating strong demand.
- Profitability varies across regions, suggesting differences in performance.
- Some transactions generate negative profit, although sub-categories remain globally profitable.
- Profit does not always scale with sales, highlighting margin differences.

## Dashboard
The Power BI dashboard provides a visual overview of:
- Sales by category
- Profit by region
- Profit by sub-category

## Business Value
This project helps identify high-performing areas and highlights opportunities to improve profitability through better cost control and product strategy.
